# ds-ml
Azure ML 을 사용하여 이탈 및 군집화 문제를 해결한다.

* 테스터 셋
데이터셋은 `game_data_utf_8_eng.csv` 를 사용한다. 해당 데이터 셋에는 이탈, 결제 여부를 포함한 유저 정보가 포함되어 있다.

* 이탈 문제
1. 데이터셋 입수
2. 데이터셋에서 필요한 칼럼만 분류
3. 데이터셋을 Train, Test 셋으로 분류
4. Two-Class Boosted Decision Tree 를 사용
5. 모델 학습 후 평가

* 결제 문제
1. 데이터셋 입수
2. 데이터셋에서 필요한 칼럼만 분류
3. 데이터셋을 Train, Test 셋으로 분류
4. Two-Class Boosted Decision Tree 를 사용
5. 모델 학습 후 평가

* 군집화 문제
1. 데이터셋 입수
2. 데이터셋에서 필요한 칼럼만 분류
3. 데이터셋을 Train, Test 셋으로 분류
4. K-Means Clustering 를 사용
5. 모델 학습 후 평가
